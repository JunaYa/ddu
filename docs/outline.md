# 截图工具产品大纲

## 1. 产品定位

构建一个面向开发者、设计师、产品经理、客服、技术写作者和高频办公用户的跨平台截图与轻量录屏工具。产品核心不是“保存图片”，而是快速完成视觉沟通：捕获、解释、隐藏敏感信息、复用、分享。

## 2. 目标用户

- 开发者：截取 bug、日志、UI 状态、代码片段，快速标注和分享到 issue、PR、IM。
- 设计师：精确量距、取色、对比前后状态、制作干净的展示截图。
- 产品与 QA：编写复现步骤、编号标注、生成测试证据。
- 客服与运营：截取用户问题、打码隐私、生成说明图或短视频。
- 技术写作者：滚动截长页、步骤编号、导出到文档或知识库。

## 3. 产品原则

- 本地优先：截图、历史、OCR、隐私打码默认在本机完成。
- 快速闭环：从快捷键到复制/保存/分享不超过 2 次确认。
- 可解释：标注、打码、裁剪和上传动作有可回溯状态，不做不可见的自动处理。
- 可配置但不打扰：高级用户能配置工作流，新用户默认路径足够好。
- 平台尊重：遵守各平台权限模型和视觉习惯，不为了统一而破坏系统体验。
- 渐进增强：Wayland、系统音频、AI、云分享在不可用时明确降级。

## 4. 核心体验闭环

1. 用户按快捷键或点击托盘/菜单栏图标。
2. 应用进入捕获 overlay，支持区域、窗口、全屏、滚动、录屏等模式。
3. 捕获成功后展示浮层，用户可复制、保存、拖放、上传、固定到屏幕或进入编辑器。
4. 编辑器以非破坏性图层保存标注，支持导出合成图。
5. 历史库记录文件、来源、OCR 文本、标签、上传状态和动作日志。
6. 分享模块根据规则输出到剪贴板、文件、云端、S3、WebDAV、FTP、脚本或第三方应用。

## 5. 能力地图

### 捕获

- 区域、固定区域、上次区域、全屏、当前屏幕、活动窗口、鼠标下窗口。
- 延时捕获、包含/排除光标、冻结屏幕、选区放大镜、尺寸和坐标提示。
- 滚动捕获，优先支持网页、文档、聊天、代码窗口。
- 多显示器、不同缩放比例、HDR/SDR、Retina/HiDPI。

### 编辑

- 裁剪、缩放、旋转、画布扩展、背景美化。
- 箭头、线条、矩形、椭圆、自由画笔、高亮、文本、气泡、步骤编号。
- 模糊、像素化、智能擦除、敏感信息打码。
- 撤销重做、图层选择、样式预设、复制粘贴标注对象。

### 记录与复用

- 捕获后浮层：复制、保存、编辑、拖放、上传、固定、删除。
- 本地历史库：最近捕获、筛选、标签、全文搜索、OCR 搜索。
- 自动命名：日期、应用名、窗口标题、计数器。
- 工作流：捕获后自动复制、保存、上传、运行脚本、打开外部应用。

### OCR 与隐私

- 本地 OCR、多语言、二维码识别。
- 邮箱、手机号、身份证号、银行卡号、URL token、API key 识别和打码建议。
- 可选 AI：摘要截图内容、生成 alt 文本、提取表格、翻译，但必须明确授权。

### 分享与协作

- 剪贴板：位图、PNG 文件、Markdown 链接、HTML、纯文本 OCR。
- 文件：PNG、JPG、WebP、PDF、SVG/JSON 项目格式、MP4、GIF。
- 上传：内置云、S3 兼容、WebDAV、FTP/SFTP、Imgur、Google Drive、OneDrive、自定义脚本。
- 链接：复制 URL、过期时间、密码保护、撤回、访问统计、团队空间。

### 录屏

- 区域、窗口、全屏录制。
- 麦克风、系统音频、摄像头画中画。
- 鼠标点击、高亮、按键展示、录制时标注。
- 简单裁剪、静音、压缩、导出 MP4/GIF。

## 6. 推荐信息架构

- 菜单栏/托盘入口：新截图、新录屏、历史、设置、最近捕获。
- 捕获 overlay：模式切换、选区、窗口高亮、尺寸输入、开始/取消。
- 捕获后浮层：最近捕获缩略图、快速动作、更多菜单。
- 编辑器：画布、工具栏、样式栏、对象属性、导出按钮。
- 历史库：时间线、搜索、筛选、标签、上传状态、批量操作。
- 设置：快捷键、保存位置、命名规则、工作流、上传器、隐私、权限、更新。

## 7. 分阶段路线图

### Phase 1: 本地截图 MVP

- 捕获区域、窗口、全屏、延时和上次区域。
- 捕获后浮层支持复制、保存、编辑和拖放。
- 编辑器支持基础标注、裁剪、模糊、撤销重做。
- 设置支持快捷键、保存路径、文件命名、开机启动。

验收标准：用户能在三大平台完成常见截图和标注，且无需登录、无需云端。

### Phase 2: 历史、OCR 与隐私

- 本地历史库和搜索。
- 本地 OCR、二维码识别、复制文本。
- 邮箱、手机号、密钥等敏感信息的检测和一键打码。
- 固定截图到屏幕、重复捕获、自动动作。

验收标准：产品从一次性工具升级为可复用工作台，隐私处理可被信任。

### Phase 3: 分享与高级工作流

- S3/WebDAV/FTP/Imgur/自定义脚本上传。
- 链接过期、密码、撤回、复制 Markdown。
- 工作流规则：按模式、应用、标签触发自动动作。
- 团队或项目空间作为可选能力。

验收标准：开发者和团队可以把截图嵌入 issue、PR、文档、客服流程。

### Phase 4: 录屏、GIF 与 AI 增强

- 区域/窗口/全屏录制，支持麦克风、系统音频、摄像头。
- 鼠标点击、按键显示、录制中标注、基础裁剪。
- GIF 导出和视频压缩。
- 可选 AI：摘要、翻译、alt 文本、表格提取、智能裁剪建议。

验收标准：覆盖异步沟通的图片和短视频场景，同时保持本地优先和明确授权。

## 8. 非目标

- 第一版不做完整视频剪辑器。
- 第一版不强制用户使用云服务或账号。
- 第一版不承诺绕过系统禁止截图的 DRM 或受保护内容。
- 第一版不追求 ShareX 级别的所有上传目的地，应先提供稳定插件接口。

## 9. 关键技术风险

- macOS 屏幕录制权限会影响截图、录屏、窗口枚举和音频采集。
- Windows HDR、DPI 缩放、UWP/Win32 窗口边界需要专项测试。
- Linux Wayland 不允许任意应用静默截图，portal 可能弹系统确认框；全局快捷键也可能不可用。
- 滚动截图依赖内容稳定性，动画、懒加载、虚拟列表和非纯色背景会影响拼接。
- OCR 和 AI 处理截图时必须有清晰的隐私边界。
