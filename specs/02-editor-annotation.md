# Spec 02: 编辑与标注

## 目标

提供快速、清晰、可撤销的截图编辑体验。编辑器应以非破坏性图层模型保存标注，导出时再合成位图，保证用户可以随时调整、隐藏、复制或删除标注对象。

## 范围

包含：

- 画布、缩放、拖拽、裁剪、扩展画布。
- 常用标注工具、样式预设、撤销重做。
- 模糊、像素化、聚光灯、高亮、步骤编号。
- 背景美化和导出渲染。

不包含：

- OCR 识别和智能打码逻辑，见 [Spec 04](./04-ocr-privacy-ai.md)。
- 历史库保存策略，见 [Spec 03](./03-history-workflow.md)。

## 用户故事

- 作为产品经理，我可以用箭头、矩形和文字快速说明问题。
- 作为 QA，我可以用步骤编号标出复现路径。
- 作为客服，我可以模糊或像素化隐私信息。
- 作为设计师，我可以给截图加背景、圆角和阴影，用于分享。
- 作为开发者，我可以复制一组标注样式并复用到下一张截图。

## 编辑器结构

编辑器分为五层：

1. 原始截图层：只读，保留原图。
2. 裁剪和画布层：定义可见区域、背景、边距、圆角、阴影。
3. 标注图层：箭头、形状、文字、步骤编号、贴纸等矢量对象。
4. 隐私处理层：模糊、像素化、擦除、智能打码。
5. 导出合成层：按目标格式渲染 PNG/JPG/WebP/PDF。

## 功能需求

### EDT-001 基础画布

- 支持 25%、50%、100%、适应窗口缩放。
- 支持触控板和鼠标滚轮缩放。
- 支持拖拽平移。
- 显示图像尺寸、缩放比例和文件大小估算。
- 大图加载时显示渐进预览，不阻塞 UI。

### EDT-002 裁剪和尺寸

- 支持自由裁剪、固定比例、输入宽高。
- 支持画布扩展和自动留白。
- 支持导出前 resize。
- 裁剪操作必须可撤销。

### EDT-003 标注工具

MVP 工具：

- 选择/移动。
- 箭头，支持直线和弯曲。
- 线条。
- 矩形和椭圆。
- 自由画笔。
- 高亮笔。
- 文本。
- 气泡/Callout。
- 步骤编号。
- 模糊。
- 像素化。
- 裁剪。

增强工具：

- 聚光灯。
- 放大镜 callout。
- 贴纸。
- 水印。
- 智能擦除。
- 图片叠加。
- 前后对比 GIF。

### EDT-004 样式系统

每个标注对象支持：

- 颜色、透明度、线宽、填充。
- 字体大小、字重、对齐。
- 箭头端点样式。
- 阴影和描边。
- 手绘风格或规整风格。

必须提供最近使用样式和全局默认样式。样式面板应跟随选中对象变化。

### EDT-005 步骤编号

- 点击画布自动添加递增编号。
- 用户可设置起始数字。
- 删除中间编号后，可选择保持编号或重新编号。
- 支持圆形、方形和纯文本三种样式。

### EDT-006 隐私处理

- 模糊和像素化必须作用于独立图层，用户可调整强度。
- 导出时隐私处理必须烘焙进位图，避免分享可逆图层。
- 对于项目格式，可选择是否保留隐私图层的可编辑信息；默认不保留原始隐私区域。

### EDT-007 背景美化

支持独立的 Beautify 模式：

- 背景色、渐变、图片背景。
- 自动边距和平衡。
- 圆角、阴影、窗口框。
- 预设比例：原始、1:1、4:3、16:9、社交媒体常用尺寸。
- 可保存用户预设。

### EDT-008 撤销重做

- 所有编辑操作必须进入 undo stack。
- 支持 Ctrl/Cmd + Z 和 Ctrl/Cmd + Shift + Z。
- 支持对象复制、粘贴、重复、对齐和删除。
- 关闭编辑器前如有未保存变更，提供明确选择：保存、丢弃、取消。

## 数据模型

```ts
interface EditorDocument {
  id: string;
  sourceCaptureId: string;
  baseImagePath: string;
  canvas: CanvasSettings;
  layers: EditorLayer[];
  createdAt: string;
  updatedAt: string;
}

interface EditorLayer {
  id: string;
  type:
    | "arrow"
    | "line"
    | "rect"
    | "ellipse"
    | "freehand"
    | "highlight"
    | "text"
    | "callout"
    | "step"
    | "blur"
    | "pixelate"
    | "magnifier"
    | "image";
  bounds: Rect;
  style: LayerStyle;
  payload: Record<string, unknown>;
  locked: boolean;
  visible: boolean;
}
```

## 验收标准

- 用户能在 10 秒内完成“截图、箭头、文字、复制”。
- 所有标注对象可选中、移动、删除、撤销。
- 导出的 PNG 和剪贴板图片与编辑器预览一致。
- 模糊和像素化在导出后不可逆。
- 8K 长图可以打开并进行基础标注，不应导致应用无响应。
- 深色/浅色系统主题下工具栏和画布对比清晰。

## 测试建议

- 单元测试：图层序列化、undo/redo、样式继承、导出尺寸。
- 视觉回归：常用工具渲染、不同缩放比例、深浅主题。
- 手动测试：大图、长图、透明背景、中文/英文文本、Retina 导出。
